## [과제 01] TIA(Today I Ate)
하루동안 먹은 식단을 기록하는 CLI 프로그램입니다.

## 1. 실행 파일
실행 파일은 tia.py입니다.

## 2. 주요 기능 및 명령어
터미널에 입력해 사용 가능합니다.

### Step 1. 사용자 이름 입력
가장 먼저 사용자의 이름을 입력받습니다.

### Step 2. 섭취 항목별 명령어
- 'eat' : 밥 또는 간식
- 'water' : 물
- 'coffee' : 커피

### Step 3. 입력 방법
선택한 명령어 따라 세부사항을 입력합니다.
- 'eat' : [음식 이름]을 텍스트로 입력
- 'water' : [마신 잔 수]를 숫자로 입력
- 'coffee' : [마신 잔 수]를 숫자로 입력
